# upbit-client

비트코인 시세를 가져오기 위한 클라이언트입니다


### clone

```bash
https://github.com/MPPULAB/upbit-client.git
cd upbit-client
```

### set `.env`
```bash
vim .env
```

```.env
ACCESS_TOKEN=<access_token>
SECRET_TOKEN=<secret_token>
```

### run

```bash
go mod tidy
go run main.go
```

or 

```
./main
```
